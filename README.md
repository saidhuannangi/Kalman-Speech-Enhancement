# 🎙️ Speech Enhancement using Kalman Filter

## 📌 Project Overview

This project focuses on enhancing speech signals corrupted by noise using the Kalman filtering technique. The Kalman filter provides optimal estimation of clean speech from noisy observations.

## 🎯 Objectives

* To reduce noise in speech signals
* To implement Kalman filtering in MATLAB
* To evaluate performance using SNR

## ⚙️ Tools Used

* MATLAB
* Signal Processing Toolbox

## 📊 Methodology

1. Input speech signal
2. Add noise
3. Apply Kalman filter
4. Analyze performance (SNR)

## 📈 Results

* Improved speech quality
* Increased Signal-to-Noise Ratio

## 📁 Files Included

* kalman_speech.m
* speech.wav
* report.pdf

## 📌 Conclusion

Kalman filtering effectively enhances speech signals in noisy environments and provides better estimation compared to basic filtering techniques.
